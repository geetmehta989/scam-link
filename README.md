# Scam Link - Valentine Page

A cute Valentine's Day page with an unclickable NO button!

## Quick Deploy to GitHub Pages (2 minutes)

1. **Create a GitHub repository:**
   - Go to https://github.com/new
   - Name it `scam-link` (or any name you prefer)
   - Make it **Public**
   - Click "Create repository"

2. **Push this code:**
   ```powershell
   git remote add origin https://github.com/YOUR_USERNAME/scam-link.git
   git branch -M main
   git push -u origin main
   ```
   (Replace `YOUR_USERNAME` with your GitHub username)

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click **Settings** â†’ **Pages**
   - Under "Source", select **main** branch and **/ (root)** folder
   - Click **Save**

4. **Your site will be live at:**
   - `https://YOUR_USERNAME.github.io/scam-link/scam-link.html`
   - Or `https://YOUR_USERNAME.github.io/scam-link/` (if you rename to index.html)

## Alternative: Use Netlify Drop

1. Go to https://app.netlify.com/drop
2. Drag and drop the `scam-link.html` file
3. Get instant live URL!
